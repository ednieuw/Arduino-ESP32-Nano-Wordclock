``` Hoe werkt het?
Dit tool genereert een .bat uploadscript dat via een PowerShell-brug wordt aangeroepen.
Klik op Upload Firmware → een .bat bestand wordt gedownload.
Dubbelklik dat bestand om de upload te starten.

arduino-cli pad aanpassen:
Als Arduino IDE 2 geïnstalleerd is, staat de CLI hier:
C:\Users\%USERNAME%\AppData\Local\Arduino15\...
of in de Arduino IDE map zelf.

Direct uploaden:
Gebruik de gegenereerde batch als basis voor automatisering. ```
